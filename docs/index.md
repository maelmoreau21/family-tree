# API Family Tree (FR)

Bienvenue dans la documentation en français. Cette page décrit les composants principaux et donne des exemples d'utilisation simples.

- f3Chart : classe principale pour créer et gérer l'arbre
- f3Card : rendu HTML des cartes
- f3EditTree : outils d'édition (forms, validation, historique)

Voir les exemples et la documentation complète dans la version anglaise si besoin.
